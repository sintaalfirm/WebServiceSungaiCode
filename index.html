di Sepanjang Sungai Code

STUDI KASUS : Jembatan Wreksodiningrat - Asrama Haji


Selamat datang! Jelajahi potensi risiko bencana banjir dan longsor sepanjang Sungai Code melalui peta interaktif. Temukan informasi banjir dan longsor, peta interaktif, serta solusi mitigasi untuk melindungi dari ancaman bencana alam. Mari bersama-sama membangun kesadaran dan kesiapsiagaan dalam menghadapi risiko bencana banjir dan longsor.

Mulai!

Kenapa Perlu Dipetakan?

Pemetaan risiko banjir dan longsor menjadi hal yang sangat penting karena dilatar belakangi oleh sebagai berikut.

Indonesia Rawan Terhadap Bencana Alam

Di Indonesia, bencana alam yang sering terjadi adalah bencana banjir dan tanah longsor. Umumnya kedua bencana ini sering disebabkan karena ulah manusia yang merusak alam.

Peran Penting Sungai Code

Sungai code memiliki peranan penting di Provinsi DI Yogyakarta. Peran sungai Code sangat penting karena kedekatannya dengan lokasi mata pencaharian masyarakat sehingga menyebabkan terjadinya pembangunan infrastruktur di sekitar sungai Code.

Pembangunan Infrastruktur Merusak Lingkungan

Pembangunan infrastruktur ini terkadang tidak memperhatikan lingkungan sehingga menyebabkan penurunan tinggi tebing sungai Code yang dapat menimbulkan kawasan sungai Code utara daerah rawan terjadi bencana banjir dan tanah longsor.

Pentingnya Peta Risiko Banjir dan Longsor

Oleh karena itu, perlu adanya pemetaan kawasan risiko bencana banjir dan tanah longsor di bantaran Sungai Code untuk mitigasi bencana, peringatan dini, peningkatan kesadaran masyarakat, dan perencanaan pembangunan yang berkelanjutan.

Tujuan Pemetaan Risiko Banjir dan Longsor

Membantu mengidentifikasi faktor-faktor terjadinya bencana banjir dan tanah longsor.

Meningkatkan kesadaran tentang bahaya banjir dan tanah longsor, serta cara untuk mengidentifikasi gejala awal dan peringatan diri.

Memberikan informasi tentang cara mengurangi dampak banjir dan tanah longsor jika bencana tersebut terjadi.

Meningkatkan kesiapsiagaan dalam menghadapi banjir dan tanah longsor.
